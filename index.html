Mengenal HTML, CSS, JAVASCRIPT dan TAILWIND

HTML

HTML adalah fondasi dari semua halaman web. Ini bukanlah bahasa pemrograman, melainkan bahasa markup yang digunakan untuk menstrukturkan konten. Dengan HTML, kita mendefinisikan elemen-elemen seperti judul, paragraf, gambar, dan tautan. Tanpa HTML, browser tidak akan tahu bagaimana cara menampilkan teks atau memuat gambar. Singkatnya, HTML adalah tulang kerangka website.

CSS

Jika HTML adalah kerangka, maka CSS adalah segala sesuatu yang membuatnya terlihat menarik. CSS digunakan untuk mengatur tampilan visual—mulai dari warna teks, jenis huruf, spasi antar elemen, hingga tata letak halaman yang kompleks. Dengan CSS, kamu bisa mengubah website yang tadinya polos menjadi sebuah karya seni digital yang indah dan responsif di berbagai perangkat. CSS adalah desainer interior untuk website.

JAVASCRIPT

Sekarang website kita sudah punya struktur (HTML) dan tampilan (CSS). Lalu bagaimana membuatnya "hidup"? Di sinilah JavaScript berperan. JavaScript adalah bahasa pemrograman yang memungkinkan kita menambahkan interaktivitas, seperti slider gambar, formulir yang divalidasi secara real-time, pop-up, atau bahkan animasi yang kompleks. JavaScript adalah otak yang membuat website merespons tindakan pengguna.

TAILWIND

Tailwind CSS adalah sebuah framework CSS yang unik. Alih-alih menyediakan komponen siap pakai seperti tombol atau kartu, Tailwind memberikan utility classes—kelas-kelas kecil yang bisa digabungkan langsung di dalam HTML untuk mendesain elemen. Contohnya, daripada menulis kode CSS terpisah untuk membuat teks tebal dan berwarna biru, dengan Tailwind kita cukup menambahkan kelas seperti class="font-bold text-blue-500". Ini sangat mempercepat proses styling dan menjaga konsistensi desain. Tailwind adalah jalan pintas cerdas untuk menata gaya website.

Tata Cara Membuat Website Sederhana

1

Persiapkan Software yang Akan Dipakai

Sebelum memulai perjalanan coding, kita perlu menyiapkan alat-alat yang akan digunakan. Software yang dapat dipakai untuk ini adalah Visual Studio Code.

2

Buat Folder Proyek

Sebelum kita mulai coding, kita perlu membuat folder serta file-file yang akan dipakai dalam proyek website. File yang dibutuhkan adalah index.html, styles.css, dan script.js.

3

Membuat Struktur Website Memakai HTML

HTML dipakai untuk membangun fondasi dari website. Contoh dari pemakaiannya adalah sebagai berikut.

4

Tambahkan Style Dengan CSS

Setelah membuat fondasi kita tambahkan style untuk mengembangkan visual dari website kita. Berikut adalah contoh penambahan visual di dalam CSS.

5

Tambahkan Fitur Dengan JAVASCRIPT

JAVASCRIPT dipakai untuk menambahkan fitur ke dalam website. Berikut ini adalah contoh sederhana dari fitur JAVASCRIPT.

6

Website Anda Selesai

Dengan tahap sebelumnya anda bisa membuat suatu website sederhana. Anda bisa melihat hasil website anda dengan memilih opsi "open in default browser" atau bisa membuka file HTML secara langsung di file explorer anda.
